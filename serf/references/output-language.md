# Output Language

Choose output language in this order:

1. Use the language explicitly requested by the user or task package.
2. Otherwise, use the language of the launch instruction or current conversation.
3. If the target report is meant for lord and the task package does not specify a language, use the task package language.

Keep literal fields unchanged: status values, file paths, command names, schema keys, task IDs, versions, and code identifiers.

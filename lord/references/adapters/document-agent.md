# Document Agent Adapter

Use for agents that process Word files, PDFs, papers, reports, slides, or other documents.

## Add to the Task Package

- Source filename and version.
- Output filename.
- Whether original formatting must be preserved.
- Whether section structure may change.
- Language, terminology, and citation-style requirements.
- Whether tracked changes, comments, or a change list are required.
- Whether a readable text handoff must also be generated.

## Add to the Handoff

- File version actually processed.
- Modification scope.
- Summary of structure, text, and formatting changes.
- Unprocessed content.
- Output filename.
- Locations requiring human review.
